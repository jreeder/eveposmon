EVEPOSMon - EVE Starbase Manager

I. First Run
Upon running the program please goto Utilities->API Keys, and enter your FULL API Keys.  After this you will need to select the character that owns the starbases you would like to monitor.  After entering your API keys you can select starbases by using the Utilities->Select Starbases menu.
